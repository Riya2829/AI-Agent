# 🤖 AI File Assistant

An AI-powered conversational assistant that helps users search, filter, and discover files from Google Drive using natural language queries.

---

## 🚀 Features

- Conversational AI-based file search
- Search files using natural language
- Filter by:
  - PDFs
  - Images
  - Videos
  - Folders
- Google Drive API integration
- Streamlit frontend
- FastAPI backend
- Clickable file links

---

## 🛠️ Tech Stack

- Python
- FastAPI
- Streamlit
- LangChain
- Groq LLM
- Google Drive API
- Render Deployment
